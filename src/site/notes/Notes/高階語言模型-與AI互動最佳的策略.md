# 語言是人類認知系統的操作介面
1. 怎麼說話，就怎麼理解世界。
2. 怎麼理解世界，就怎麼預測未來。
3. 怎麼預測未來，就怎麼行動。
# 語言升級的樣貌
1. 低維情緒語言，升級成[[Notes/結構語言\|結構語言]]。
2. 抱怨語言，升級成[[Notes/問題定義\|問題定義]]。
3. 混亂語言，升級成[[Notes/模型拆解\|模型拆解]]。
4. 二元對立語言，升級成[[Notes/系統分析\|系統分析]]。
# 語言升級後的成效
1. 把「我不行」，升級成「我的能力節點尚未完成」。
2. 把「我很焦慮」，升級成「我的預測模型正在面對不確定性」。
3. 把「我失敗了」，升級成「現實回傳了誤差訊號」。
4. 把「我不知道怎麼辦」，升級成「我需要重新設定[[Notes/目標函數\|目標函數]]與[[Notes/行動路徑\|行動路徑]]」。


> [!tip]
> 不再被情緒拖著走。把生命變成一套可觀察、可拆解、可修正、可更新的系統。
